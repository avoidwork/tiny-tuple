tiny-tuple

Tiny tuple for Client or Server, which extends Array.

Example

var tuple = require('tiny-tuple'),
    x = tuple(1,2,3, tuple(2,4));

x.length; // 4
x.extract(); // [1,2,3,2,4]
Object.isFrozen(x); // true

API

extract()

Returns a frozen Array of flattened Tuple items
